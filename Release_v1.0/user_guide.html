Hana_Cee's Stream Pet

Quick-start guide for streamers – install, connect, and play ✨

Twitch: https://twitch.tv/hana_cee

1. What this is

This guide turns the technical reference for Hana_Cee's Stream Pet into a step-by-step walkthrough for an average-skill OBS user. Follow it in order and you’ll have a cute, clickable pet on stream that reacts to Twitch events like follows, subs, and redeems.

You will need: a Windows/Mac PC with OBS Studio, the Stream Pet folder extracted somewhere permanent (e.g. C:\StreamStuff\HanaCeePet), and your Twitch account.

2. Get the files ready

  1. Extract the download. Unzip the folder you got for the pet. Put it somewhere you won’t move later (OBS doesn’t like it when the file disappears).
  2. Remember the folder. Inside you should see files like index.html and config.html. index.html is what OBS shows. config.html is where you change settings.
  3. Optional: keep your custom pet images or sounds inside the same main folder to avoid path issues.

If you move the folder later, you must update the file path in OBS.

3. Add it to OBS (the display part)

  1. Open OBS Studio.
  2. In the Sources box, click + → Browser. Name it something like HanaCee_Pet.
  3. Check Local file, then click Browse… and pick the index.html inside your pet folder.
  4. Set width and height to match your canvas (usually 1920 x 1080).
  5. Click OK. You should see the pet area appear. If it’s blank, keep going — we’ll configure it next.

You can enable “Refresh browser when scene becomes active” to force the pet to reload whenever you switch scenes – useful while testing.

4. Open the configuration panel

You can configure the pet in two ways. Use whichever is easier:

Method A – In OBS

Right-click your pet browser source → Interact. A mini browser pops up. Go to config.html from there (sometimes it’s linked in the pet).

Method B – In your normal browser

Double-click config.html in the pet folder. This is usually easier because you get full tabs, testing buttons, and can see if Twitch connects properly.

The config screen has lots of tabs, but for a basic setup you only need Pet Settings and Twitch to start.

5. Make it look like your pet

In Pet Settings (or similar) set the basics:

  • Name: what you want alerts to show (e.g. “Hana’s Buddy”).
  • Image: pick one of your included images or your own. You can use the ones you shared:
    • hana.png – chibi mascot
    • StreamPet.png – cute bird with glasses
  • Size: if it’s too big/small, change “size mode” to Pixels and try something like 300 px.
  • Position: set it to the lower-left or lower-right of your screen so it doesn’t cover your gameplay.

If OBS shows a different size than the config preview, refresh the OBS browser source.

6. Connect it to Twitch (so it reacts!)

This part lets Hana_Cee’s Stream Pet listen to your chat and events.

  1. In config.html go to the tab called Twitch (or Integration / Twitch).
  2. Type your channel name without the #. Example: hana_cee.
  3. Click the button to Authenticate or Connect to Twitch.
  4. A Twitch window opens. Log in and click Authorize. This gives the pet permission to see your events.
  5. Go back to the config page. It should say something like Connected or show your status.
  6. Find the Test area on the config page and click Test Follow or Test Sub. Your pet in OBS should react.

If it says “redirect URL mismatch,” double-check the URL field in the Twitch tab matches what you put in your Twitch developer app. That’s a common hiccup.

7. How to use it during stream

Once it’s in OBS and connected, most things are automatic:

  • New follow / sub / bits: pet plays the animation you picked.
  • Chat commands: viewers can type things like !pet, !pet dance, or whatever is listed in the Commands tab.
  • You: you can trigger tests from the config anytime, even while live.

If you want to add your own command (e.g. !birb to make the bird spin):

  1. Open the Commands tab.
  2. Click Add Command.
  3. Set the command name (without ! usually the pet adds that).
  4. Pick the animation/state you want.
  5. Save. Test from chat.

8. Optional: extras you can explore later

These are in the technical guide but you don’t have to set them up on day one:

  • Spotify reactions: pet dances differently based on the music.
  • Seasonal modes: automatic Halloween/Christmas themes.
  • Growth / XP: pet “levels up” when your chat interacts.
  • StreamElements / Streamlabs: connect their tokens to make donations trigger the pet.

Everything saves to your browser’s local storage, so export your settings once you like them.

9. Quick troubleshooting

Pet is blank in OBS

  • Right-click the source → Properties → make sure it points to index.html in the correct folder.
  • Click Refresh cache of current page.
  • Make sure width/height aren’t 0.

Settings don’t stick

  • Click the Save / Apply button in config.
  • Try opening config.html in Chrome/Edge if another browser blocks local saving.
  • Use the Export option to keep a backup.

Twitch won’t connect

  • Re-authenticate in the Twitch tab.
  • Make sure the Redirect / Callback URL in Twitch matches what the pet asks for.
  • Test again with the built-in “Test Follow” buttons.
